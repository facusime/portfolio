Proyecto 2

Descripción del Proyecto "Matemática 3 con Python":
Este proyecto fue desarrollado como parte del curso de Matemática 3 y utiliza el lenguaje de programación Python para abordar dos tareas fundamentales relacionadas con conceptos matemáticos. El objetivo principal de este proyecto es aplicar conceptos teóricos de matemáticas a través de la programación y la resolución de problemas específicos. Aquí se describen las dos partes del proyecto: Generación de Matriz de Incidencia para Grafo No Dirigido: En esta parte del proyecto, se ha creado un programa que genera una matriz de incidencia arista-vertice de tamaño 7x6 de forma aleatoria. La matriz de incidencia arista-vertice es una herramienta utilizada en la teoría de grafos para representar relaciones entre aristas y vértices en un grafo no dirigido. El programa imprime la matriz generada y calcula los siguientes valores: Número de aristas en el grafo. Número de lazos en el grafo (aristas que conectan un vértice consigo mismo). Grado de cada vértice, es decir, el número de aristas incidentes en cada vértice. Determinación de Reflexividad y Simetría en Relación Binaria: En esta parte del proyecto, se ha creado un programa que se enfoca en una relación binaria definida sobre un conjunto finito A. El programa determina si la relación binaria es reflexiva y/o simétrica. Una relación binaria se considera reflexiva si cada elemento en el conjunto A está relacionado consigo mismo. Una relación binaria es simétrica si para cada par de elementos (a, b) en la relación, el par (b, a) también está presente en la relación.
